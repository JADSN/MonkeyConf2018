Shell es un nuevo contenedor para aplicaciones Xamarin.Forms que ofrece varios beneficios:

* Interfaz de usuario con temas por defecto, comenzando con Material Design.
* Navegación moderna lista para usar a través de un control de Flyout.
* Navegar por rutas.
* Diseño unificado.

En esta sesión, vamos a conocer todas las características fundamentales de Shell, ¿te apuntas?
